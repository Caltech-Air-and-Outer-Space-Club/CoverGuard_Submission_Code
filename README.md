# CoverGuard_Submission_Code
Official software submission for NASA Space to Soil Challenge. CoverGuard is a SmallSat framework for onboard cover crop failure diagnosis. This repo contains (1) NASA OPERA/HLS ground-disturbance monitor, (2) Onboard multispectral inference using LightGBM for pre-symptomatic failure detection, and (3) Priority-based adaptive downlink simulation.
